Formato: Intensivo | 5 semanas | 2 hrs/semana (Live-Online) + Retos asíncronos.

About
Este curso intensivo, avalado por el CUGDL y el Google Developer Group (GDG) Jalisco, está diseñado para llevarte de cero a la creación de tu primera aplicación web interactiva con Python en solo 5 semanas. Olvídate de la teoría interminable; este es un curso 100% práctico y orientado a la industria. Configuraremos tu computadora como la de un desarrollador profesional, aprenderás las bases de Python, manipularás datos reales y construirás un portafolio usando las mismas herramientas (y asistentes de IA) que los ingenieros de software utilizan en su día a día. Ya sea que estudies Ciberseguridad, IA, Creatividad Digital, Finanzas o Biomedicina, la programación será tu superpoder.

By the end, you’ll have:
•	Un entorno de desarrollo profesional: Dominio básico de VS Code, entornos virtuales y control de versiones (Git/GitHub).
•	Bases sólidas en Python: Comprensión práctica de variables, estructuras de control, funciones y lógica algorítmica.
•	Tu primera Web App desplegada: Un proyecto final público en la nube (Streamlit Community Cloud) que resuelve un problema usando datos.
•	Flujo de trabajo con IA: Experiencia integrando asistentes de código de IA (como Gemini Code Assist/Copilot) para acelerar tu aprendizaje, debugear errores y escribir código como un profesional del 2026.
•	Certificación: Badge digital oficial de CUGDL x GDG Jalisco para destacar tu perfil en LinkedIn, y la validación de créditos (equivalente a 1 taller).










Content (Syllabus)

Week 1: El Entorno Real y Primeros Pasos (Setup & Basics)
El objetivo no es solo programar, es programar como un profesional.
•	Instalación y Setup: Python, VS Code, Git, y creación de tu cuenta en GitHub.
•	Tu primera línea de código: Hola Mundo, variables, y tipos de datos (Strings, Integers, Floats, Booleans).
•	Operadores y Lógica: Condicionales (if/else) para la toma de decisiones.
•	Hands-on: Creación de un repositorio en GitHub, escritura de un script básico y tu primer commit y push.

Week 2: Estructuras y Lógica (Core Programming)
Cómo manejar y procesar información eficientemente.
•	Estructuras de Datos: Listas y Diccionarios. (Ejemplos aplicados: una lista de IPs para ciberseguridad, un diccionario de pacientes para biomedicina).
•	Iteraciones (Bucles): for y while. Automatizando tareas repetitivas.
•	Funciones: Creando bloques de código reutilizables (Mantenlo S.O.L.I.D. y limpio).
•	Hands-on: Construir un pequeño programa de terminal (ej. una calculadora financiera básica o un validador de contraseñas) y subir la actualización a GitHub.

Week 3: El Poder de los Datos (Pandas & APIs)
Python brilla por sus librerías. Aquí empieza la magia.
•	Introducción a entornos virtuales: venv. ¿Por qué aislar proyectos?
•	Librerías externas (pip): Instalación y uso de Pandas.
•	Manipulación de Datos: Lectura de archivos .csv, exploración rápida de DataFrames, filtrado y limpieza básica.
•	Hands-on: Cargar un dataset real (ej. históricos de criptomonedas o registros médicos anonimizados) y extraer 3 conclusiones clave (promedios, máximos, conteos) mediante código.

Week 4: Construyendo el Frontend (Introducción a (COMO EJEMPLO, podría ser cualquier otra biblioteca) Streamlit)
Tus scripts dejan la terminal y se vuelven interactivos.
•	¿Qué es Streamlit?: Transformando scripts de datos en Web Apps en minutos.
•	Componentes UI: Títulos, textos, botones, sliders y selectbox.
•	Interactividad: Conectando los inputs del usuario con el análisis de Pandas de la semana anterior.
•	Hands-on: Integrar el análisis de la Semana 3 en una interfaz gráfica local donde el usuario pueda filtrar los datos dinámicamente con un slider o dropdown.

Week 5: Deployment y Presentación (Showcase Day)
Si no está en internet, no existe. Preparándonos para el mundo laboral.
•	Deployment: Subir la aplicación de Streamlit a la nube de forma gratuita (Streamlit Community Cloud) directamente desde GitHub.
•	Buenas prácticas finales: Documentación (README.md), código limpio y cómo hablar de tu proyecto en una entrevista.
•	Showcase: Los estudiantes comparten sus links públicos en el chat.

•	Cierre: Instrucciones para reclamar el badge de LinkedIn y el trámite de los 20 créditos en CUGDL.
